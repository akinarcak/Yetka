# Yetka Security Policy

Yetka is an open-source Privileged Access Management (PAM) platform. We take
security issues seriously and appreciate responsible disclosure.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately through the
[Yetka security advisory form](https://github.com/akinarcak/Yetka/security/advisories/new).
Do not open a public issue for an unpatched vulnerability or include secrets,
credentials, or personal data in a report.

Please include:

- Affected version or commit
- Clear reproduction steps
- Expected and observed behavior
- Any relevant logs or proof of concept, with sensitive values redacted

We will acknowledge valid reports, investigate them, and coordinate a fix and
disclosure timeline with the reporter.

## Güvenlik Politikası

Yetka, açık kaynak bir Ayrıcalıklı Erişim Yönetimi (PAM) platformudur. Güvenlik
sorunlarını ciddiye alıyor ve sorumlu bildirimleri destekliyoruz.

Güvenlik açığı şüphenizi [Yetka güvenlik bildirimi formu](https://github.com/akinarcak/Yetka/security/advisories/new)
üzerinden özel olarak gönderin. Yaması yayınlanmamış açıkları herkese açık
issue olarak bildirmeyin; rapora parola, anahtar veya kişisel veri eklemeyin.
